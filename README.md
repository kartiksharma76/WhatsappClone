# 📖 Overview

**WhatsApp Clone** is a production-ready, real-time messaging application built with **Spring Boot**, **React**, **WebSocket/STOMP**, and **MySQL**.

The platform supports one-to-one messaging, group conversations, media sharing, typing indicators, delivery/read receipts, online presence tracking, and secure JWT authentication.

Designed with enterprise-grade architecture, the project demonstrates modern backend development, scalable WebSocket communication, and responsive frontend design.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/whatsapp-clone.git
```

---

## Docker Deployment

```bash
docker compose -f docker/docker-compose.yml up --build
```

Application

```
Frontend
http://localhost:3000

Backend
http://localhost:8080

Swagger
http://localhost:8080/swagger-ui.html
```

---

## Manual Setup

### Backend

```bash
cd backend

mvn spring-boot:run
```

---

### Frontend

```bash
cd frontend

npm install

npm run dev
```
